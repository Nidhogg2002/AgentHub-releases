# 🤖 Agent Hub

> **多 AI 智能体一站式桌面管理与实时监控平台**
> 
> 集中管理、聚合并调优你本地的 Claude Code、Codex 等多种终端智能体。提供统一会话历史、运行轨迹监控及流畅的一键安装体验。

---

## 📥 下载与安装 (Windows 平台)

本软件暂仅提供 Windows 系统的打包支持，请按照以下步骤下载安装：

1. 打开本页面右侧的 **[Releases (发布版)](https://github.com/Nidhogg2002/AgentHub-releases/releases)** 页面。
2. 找到最新的版本（例如 `v1.0.0`），在下方的 **Assets** 列表中点击下载安装包：**`agent-hub-1.0.0-setup.exe`**。
3. 双击运行下载的安装程序，按照提示选择安装路径。
4. 安装完成后，在桌面上双击 **Agent Hub** 即可启动。

---

## 🌟 核心功能

* 🔍 **自动扫描检测**：一键检测本地系统是否安装有 Claude Code 等 AI 智能体。
* 💬  **统一会话视图**：跨 Agent 归档与检索你的历史会话，支持会话的重命名、搜索与删除。
* 📂 **项目会话聚合**：按代码仓库的物理路径聚合你的会话，并提供便捷的项目文件树预览。
* 👁️ **实时运行追踪 (Inspector)**：专业级图形追踪控制台，实时观测 Agent 思考逻辑、Tool 调用轨迹及 Subagent 状态。
* 🐚 **嵌入式终端**：内置集成终端，不离开软件即可与智能体直接对话并拉起原生命令行。
* 🧠 **跨 Agent 记忆同步**：内置 SQLite 跨智能体共享本地知识记忆。

---

## ⚙️ 系统要求与准备

运行前，请确保你的电脑上已经安装有以下任何一款 AI 终端工具，并在本地完成登录配置：
* **Claude Code** (由 Anthropic 推出的终端 AI 开发助手)
* **Codex / AntiGravity**

*软件启动后将自动检索并在主页为您呈现本地已安装的 AI 智能体！*
